# ☕ Mokavava — Coffee House Website

A fully responsive, single-page coffee shop website built with pure HTML, CSS, and JavaScript.

## Features

- Smooth-scroll single-page layout
- Sticky navbar with scroll effect
- Animated hero section
- Filterable product menu with 12 items
- Interactive shopping cart sidebar
- Scroll-reveal animations
- Contact form
- Mobile-responsive design
- No frameworks, no dependencies — just vanilla web tech

## Project Structure

```
mokavava/
├── index.html    # Main HTML structure
├── style.css     # All styles and animations
├── script.js     # Interactive behavior
└── README.md     # This file
```

## Deploy in 1 Click

### GitHub Pages (Free)

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Under "Source", select **main branch / root**
4. Click Save — your site is live at `https://yourusername.github.io/repository-name`

### Netlify (Free)

1. Go to [netlify.com](https://netlify.com)
2. Drag and drop this folder onto the Netlify dashboard
3. Your site is live instantly at a generated URL

### Vercel (Free)

1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` inside this folder
3. Follow the prompts — live in seconds

### Cloudflare Pages (Free)

1. Go to [pages.cloudflare.com](https://pages.cloudflare.com)
2. Connect your GitHub repo and select the root directory
3. No build command needed — deploy directly

## Local Development

Just open `index.html` in your browser — no build step required.

Or use a simple local server:

```bash
# Python 3
python -m http.server 8000

# Node (npx)
npx serve .
```

Then visit `http://localhost:8000`
